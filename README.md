# SAP ABAP Development Portfolio

Showcasing my transition from Functional SAP Consulting (MM/SD) to ABAP Development. This portfolio demonstrates my foundational skills in ABAP programming, data extraction, and modern S/4HANA basics. 

As a developer with functional experience, my goal is to bridge the gap between business requirements and clean technical implementations.

## Project 1: Standard ALV Reporting (Application Layer)
**File:** `z_po_overview.abap`
* **Objective:** Developed a custom ALV report to extract and display Purchase Order data.
* **Technical Highlights:** Utilized ABAP Data Dictionary (DDIC), internal table operations, and standard ALV Grid functionalities for clean and accurate data presentation.
